
# Street Painter

## Description
Street Painter is a Unity game I developed in a four person team during a one week game jam sprint at my community college. Acting as the only programmer, I designed and implemented the scripts behind the project while instructing my fellow developers on the use of GitHub and Unity's scripting system.
## Role
Lead Programmer

## Features
- Player movement, animation controllers, health/UI system.
- State machine driven AI for npcs
- Physically based projectile system with object pooling for weapon system.
